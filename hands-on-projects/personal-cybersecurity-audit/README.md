## Project Overview
This project documents a personal cybersecurity audit conducted to assess the security posture,
identify potential risks, and recommend improvements for my personal computer, accounts, and daily usage
practices.

## Objective
To evaluate personal cybersecurity controls, identify security gaps, and apply basic risk
assessment and mitigation principles.

## Scope of the Audit
- Account security and authentication practices
- Device and system security
- Data protection and privacy considerations
- User behavior and security awareness

## Activities Performed
- Reviewed existing security controls
- Identified vulnerabilities and risk areas
- Assessed potential impact and likelihood
- Documented findings and recommendations

## Skills Demonstrated
- Security risk assessment
- Cybersecurity auditing fundamentals
- Threat awareness
- Security documentation and reporting
